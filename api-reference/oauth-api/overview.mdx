This is how a connected OAuth Application will interact with the OAuth Flow.

Once a valid access token is returned from the oauth/authorize or oauth/token endpoint, you can use that within the header of requests using the following format:

- key: `Authorization`
- value: `Bearer access-token-goes-here`
